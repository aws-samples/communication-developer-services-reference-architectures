# Examples

Current reference files that support the catalog, kept in the repo because they
are data rather than a standalone sample repo. These are actively maintained
(unlike the archived material under [`legacy/`](../legacy/)).

- **[notify-advanced-countries.csv](notify-advanced-countries.csv)** — countries
  in the AWS End User Messaging "advanced" tier for SMS, where sender
  registration or other requirements apply before you can send.
- **[Global SMS Planning Sheet-Public.xlsx](Global%20SMS%20Planning%20Sheet-Public.xlsx)** —
  a worksheet for planning global SMS rollouts across countries and use cases.
